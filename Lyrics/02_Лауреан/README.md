# Альбом №2. «Лауреан»

«Лауреан» — камерный альбом из трёх песен, объединённых образами моря, звёзд и ночной тишины. Каждая песня — признание в любви, но адресованное по-разному: первая — через тоску по далёкому человеку на берегах Невы, вторая — через метафору корабля, плывущего к звезде, которая оказывается любимым человеком, третья — через переживание ночного неба, где звезда Лауреан становится чем-то глубоко личным. Общий тон — тихий, созерцательный, с ощущением пространства и нежности.
