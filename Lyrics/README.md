## Как устроено

- Каждый альбом — отдельная папка `NN_Название`. Внутри песни пронумерованы по порядку трека (`01_…`, `02_…`).
- Русские альбомы — оригиналы; англоязычные (№9–11) — поэтические переложения, отдельные альбомы.
- В каждой папке альбома есть подпапка `*_OTS` с файлами доказательства времени (OpenTimestamps): хеш каждого текста закреплён в блокчейне Bitcoin, что подтверждает дату создания независимо от автора и GitHub.
